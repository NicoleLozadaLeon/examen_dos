# Calculadora de MRU y MRUA
`Esta aplicación de consola en C# tiene como objetivo proporcionar una herramienta para realizar cálculos relacionados con el Movimiento Rectilíneo Uniforme (MRU) y el Movimiento Rectilíneo Uniformemente Acelerado (MRUA).`

## Instrucciones de ejecución
1. Descargue el programa de calculadora [enlace al repositorio].
2. Abra el programa en su entorno de desarrollo
3. Ejecute el programa.

## Ejemplos de uso
### Calculadora de MRU
#### Distancia
Para calcular la distancia, ingrese el valor de velocidad, y luego ingrese el valor del tiempo.

 `d = 2 * 3`

Respuesta: 6
#### Velocidad
Para calcular la velocidad, ingrese el valor de distancia, y luego ingrese el valor del tiempo.

 `v = 6 / 3`

Respuesta: 2

#### Tiempo
Para calcular el tiempo, ingrese el valor de la distancia, y luego ingrese el valor de la velocidad.
 `t = 6 / 2`

Respuesta: 3

### Calculadora de MRUA
#### Posición final
Para calcular la posición final, ingrese el valor de posición inicial, ingrese el valor de velocidad inicial, , ingrese el valor del tiempo y luego ingrese el valor de la aceleración.

 `X = 0 + 1 * 3 + 0,5 * 1 * (3 * 3) `

Respuesta: 4, 50000
#### Velocidad final
Para calcular la velocidad final, ingrese el valor de la velocidad inicial, ingrese la aceleración, y luego ingrese el valor del tiempo.

 `v = 1 + 1 * 3`

Respuesta: 3
#### Tiempo
Para calcular el tiempo, ingrese el valor de velocidad final,, ingrese el valor de la velocidad incial y luego ingrese el valor de la aceleración.

 `t = ( 1 - 0) / 1`

Respuesta: 1
