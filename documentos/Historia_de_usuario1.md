# Historia de Usuario No. 001

## Calculadora de problemas de MRU

`Como usuario, necesito funciones de (MRU) Movimiento Rectilíneo Uniforme para poder calcular problemas de Movimiento Rectilíneo Uniforme`

## Descripción

Calculadora de la distancia, velocidad y el tiempo de Movimiento Rectilíneo Uniforme.

## Criterios de Aceptación

1. **El usuario debe poder usar una de las tres funciones**
   - Fórmula de la distancia
   -  Fórmula de la velocidad
    - Fórmula de el tiempo
2. **El usuario debe poder elegir realizar otro cálculo al finalizar el programa. **
   - Introducir un número para volver a empezar el programa
   - Introducir un número para dar por finalizado el programa.
